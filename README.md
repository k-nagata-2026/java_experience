# Javaプログラミング体験：最初の一歩

このプロジェクトは、Javaの書き換えと実行を体験するための練習用リポジトリです。

## 🚀 今日のミッション
1. このプロジェクトを自分のPCにコピー（git clone）する
2. VS Codeで開き、中の文字を書き換える
3. プログラムを実行して、自分のメッセージを表示させる！

## 📝 実行方法
1. 右上の **再生ボタン（▷）** をクリック
2. 画面下のターミナルを確認

## ⚠️ 注意
- 全角のスペースが入るとエラーになります！
- 書き換えたら `Ctrl + S` で保存するのを忘れずに。
